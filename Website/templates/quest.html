Color the graph

Given is a graph with N vertices and N directed edges, such that no vertex has more than one edge incident on it.
Write a program called "color" that determines the minimum number of colors needed to color the edges so that for each vertex, the edges incident on it have different colors.

Input

  • On the first line of the standard input, a number T is specified - the number of test cases. Following are T pairs of lines.
  • On the first line of each pair is the number N - the number of vertices for this test. On the next line are N numbers Xi, separated by spaces.
  • The number Xi written at the i-th position indicates that the graph has a directed edge from vertex Xi to vertex i, where i = 1, 2, ..., N.

Output

For each test case, the program outputs on a separate line the sought minimum number of colors.

Costraints

  • 1 ≤ Т ≤ 10
  • 2 ≤ N ≤ 100000
  • 1 ≤ Xi ≤ N
  • Xi ≠ i

Sample 1:

Input
2
4
4 1 2 3
5
4 1 2 3 4
Output
2
3
